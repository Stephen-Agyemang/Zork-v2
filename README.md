# Zork v2

Zork v2 is a text-based adventure game where players explore locations, collect items, and interact with containers in a small world inspired by the classic Zork game.

## How to Run

Compile and run the game with:

```bash
javac *.java
java Driver

**Commands**
look, go <direction>, take <item>, take <item> from <container>, drop <item>, put <item> in <container>, inventory, help, quit
